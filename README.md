## 构建步骤

目前需要结合后台项目一起使用

``` bash
# 克隆项目
git clone https://github.com/chhpt/lite-reader.git

# 进入项目目录
cd lite-reader

# 安装项目依赖（需要提前安装 yarn）
yarn

# 根据平台构建应用
yarn run build
```
